# RT 05 — Parte 1.1/2016 — PPCI Forma Completa (ROUTER)

> **Skill fatiada.** Conteúdo detalhado em `_caps/rt05-p11/`. Este arquivo é o
> **roteador**: identifique o subtema e abra o cap. **Um cap por vez.**

## Escopo no edital
Cobrada **integralmente** (078/2024 CTSP e 038/2024 CBA), **sem recorte**.
Procedimento administrativo das edificações regularizadas por **PPCI na forma
completa** (base: LC 14.376/13 + Dec. 51.803/14). Vigente desde 07/11/2016
(revogou a RT 05 Parte 01/2016).

> "Segundo a RT 05 Parte 1.1" → resposta sai **exclusivamente** desta RT.

## Regra de roteamento
1. Aplicação/definição → Cap.01 · medida → Cap.02 · trâmite/renovação → Cap.03 · responsabilidade/recurso/anexo → Cap.04.
2. **Dimensionamento técnico** de medida → RTs próprias (11/12/14/15). **PSPCI** → RT 05 P. 3.1; **existentes** → RT 05 P. 07; **dispensadas** → RT 05 P. 02 — avaliar realocar_leg.

## Pegadinhas globais
- **PPCI ≠ APPCI**; forma **completa ≠ PSPCI**.
- Trâmite **via SOL-CBMRS**; **vigência do APPCI não é perpétua**.
- Responsabilidade **tripartite**; recursos para **Análise e Vistoria**.
- Justificativa cita item da RT, nunca "alternativa X".
