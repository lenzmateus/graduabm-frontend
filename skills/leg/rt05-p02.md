# RT 05 — Parte 02/2023 — Atividades Dispensadas e de Baixo Risco (ROUTER)

> **Skill fatiada.** Conteúdo detalhado em `_caps/rt05-p02/`. Este arquivo é o
> **roteador**: identifique o subtema e abra o cap. **Um cap por vez.**

## Escopo no edital
Cobrada **integralmente** (078/2024 CTSP e 038/2024 CBA), **sem recorte**.
Requisitos das **atividades dispensadas do licenciamento** e das **atividades de
baixo risco** (LC 14.376/13 + Dec. 51.803/14). Vigente desde 03/2023.

> "Segundo a RT 05 Parte 02" → resposta sai **exclusivamente** desta RT.

## Regra de roteamento
1. Definição → Cap.01 · dispensa → Cap.02 · baixo risco → Cap.03 · procedimento/responsabilidade/anexo → Cap.04.
2. **PPCI/PSPCI** → RT 05 P. 1.1 / P. 3.1; **silos/armazéns** → RT 22; **isolamento** → RT 04; medidas técnicas plenas → RT 11/12/14/15 — avaliar realocar_leg.

## Pegadinhas globais
- **Dispensada ≠ baixo risco** (baixo risco **exige** 5 medidas).
- **200 m²** área total; **GLP ≤ 26 kg**; **subsolo ≤ 50 m²**; **F-6 nunca dispensa**.
- Dispensa não é permanente; CBMRS vistoria a qualquer momento.
- Justificativa cita item da RT, nunca "alternativa X".
